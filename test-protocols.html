Protocol Handler Test

This page tests if your browser and OS allow opening applications via "Deep Links" (URL Schemes).

Test 1: VS Code

Click the button below. If VS Code is installed, it should prompt to open.

Open VS Code (vscode://)

Requires Visual Studio Code to be installed.

Test 2: Antigravity

Click the button below. If an app registered for 'antigravity://' is installed, it should prompt to open.

Open Antigravity (antigravity://open)

Requires the Antigravity app with protocol handler registration.
